Doctome - Application Overview

Doctome is a modern, responsive, and multilingual medical practice management dashboard built with React and Tailwind CSS. It is designed to help healthcare professionals and clinic administrators manage their daily operations, monitor patient appointments, and mitigate risks such as patient no-shows.

Key Features & Modules:

1. Dashboard & Risk Assessment
- A high-level overview of daily metrics including Total Appointments, Predicted No-Shows, Confirmed Visits, and Clinic Efficiency.
- A predictive Risk Table that evaluates the "no-show" probability of scheduled patients based on factors like weather, traffic, and attendance history.
- Active Insights that alert staff to potential scheduling conflicts or weather impacts.

2. Patient Management
- A comprehensive patient directory to add, search, filter, and modify patient details.
- Displays demographics, contact info, status (Active/Inactive), and last visit dates.
- Features quick actions to view details, send SMS alerts, or export patient data.

3. Appointment Scheduling
- Manage appointments across distinct statuses (Upcoming, In Progress, Checked Out, Canceled).
- Dual viewing modes: Schedule List and Calendar View.
- Ability to create, modify, and track appointments with associated provider details.

4. Analytics & Reporting (Analytics)
- Weekly trend charts and data visualization for clinic performance.
- Metrics tracking to monitor overall clinic health and patient volume over time.

5. Internationalization & Accessibility (i18n)
- Full multilingual support built entirely from scratch without external dependencies.
- Supported languages: 
  * English (LTR)
  * French (LTR)
  * Arabic (RTL)
- Dynamic UI mirroring: The application automatically toggles between Left-to-Right (LTR) and Right-to-Left (RTL) layouts depending on the selected language, ensuring native readability for Arabic users.

6. Theme & Customization
- Integrated Theme Context supporting Light and Dark modes.
- Settings page for managing general preferences, notifications, and profile configurations.

Technical Stack:
- Frontend Framework: React (Create React App)
- Styling: Tailwind CSS
- Icons: Lucide React
- Routing: React Router
- State Management: React Context API (ThemeContext, LanguageContext, AuthContext, ToastContext)

Doctome prioritizes a streamlined user experience, ensuring that clinic staff can operate efficiently across different languages and visual preferences while maintaining a clear view of patient needs and clinic performance.

**Description de l'application (Français)**

Doctome est un tableau de bord de gestion de cabinet médical moderne, réactif et multilingue, conçu avec React et Tailwind CSS. Il est conçu pour aider les professionnels de la santé et les administrateurs de cliniques à gérer leurs opérations quotidiennes, suivre les rendez-vous des patients et atténuer les risques tels que les absences.

Fonctionnalités clés et modules :

1. Tableau de bord et évaluation des risques
- Vue d'ensemble des indicateurs quotidiens : Total des rendez-vous, Absences prévues, Visites confirmées et Efficacité.
- Tableau de risques prédictif basé sur la météo, la circulation et l'historique.
- Alertes et insights actifs.

2. Gestion des patients
- Répertoire pour ajouter, rechercher, filtrer et modifier les patients.
- Actions rapides pour les SMS ou l'exportation.

3. Gestion des rendez-vous
- Suivi des statuts (À venir, En cours, Terminé, Annulé).
- Vues en liste et calendrier.

4. Analytique
- Graphiques de tendances hebdomadaires.

5. Internationalisation (i18n)
- Anglais (LTR), Français (LTR), Arabe (RTL).
- Basculement dynamique de l'interface.

6. Thème
- Mode clair/sombre et paramètres.

Stack technique : React, Tailwind CSS, Lucide, React Router, Context API.

**نظرة عامة على التطبيق (العربية)**

دوكتوم (Doctome) هي لوحة معلومات حديثة وسريعة الاستجابة ومتعددة اللغات لإدارة العيادات الطبية، مبنية باستخدام React و Tailwind CSS. تم تصميمها لمساعدة المتخصصين في الرعاية الصحية ومديري العيادات على إدارة عملياتهم اليومية ومراقبة مواعيد المرضى وتقليل المخاطر مثل عدم حضور المرضى.

الميزات والوحدات الرئيسية:

1. لوحة القيادة وتقييم المخاطر
- نظرة عامة على المقاييس اليومية: إجمالي المواعيد، حالات الغياب المتوقعة، الزيارات المؤكدة، وكفاءة العيادة.
- جدول مخاطر تنبؤي بناءً على الطقس وحركة المرور والتاريخ.
- تنبيهات ورؤى نشطة.

2. إدارة المرضى
- دليل شامل لإضافة المرضى والبحث عنهم وتصفيتهم وتعديل بياناتهم.
- إجراءات سريعة للرسائل القصيرة أو التصدير.

3. جدولة المواعيد
- تتبع الحالات (القادمة، قيد التقدم، المكتملة، الملغاة).
- طرق عرض القائمة والتقويم.

4. التحليلات
- مخططات الاتجاه الأسبوعية.

5. التدويل واللغات (i18n)
- الإنجليزية (LTR)، الفرنسية (LTR)، العربية (RTL).
- تبديل واجهة المستخدم ديناميكيًا.

6. السمة (Theme)
- الوضع الفاتح والداكن والإعدادات.

التقنيات المستخدمة: React، Tailwind CSS، Lucide، React Router، Context API.
